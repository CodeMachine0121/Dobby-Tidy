# i18n 語言切換設計文件

**日期：** 2026-04-16  
**狀態：** 已核准，待實作  
**範疇：** Dobby frontend（React + TypeScript）

---

## 目標

為 Dobby 前端新增中文／英文雙語支援，語言偏好存在 `localStorage`，切換後全 app 即時更新，重啟後保持選擇。語言切換 UI 放在 Settings 頁面。

---

## 方案選擇

採用 **react-i18next**（`i18next` + `react-i18next`）。

選擇理由：
- App 已有動態插值需求（例如 `還剩 N 天`、`N 條規則，M 條啟用中`），react-i18next 內建插值語法可乾淨處理
- 未來如需新增第三語言，直接新增語言檔即可
- Desktop app 對 bundle 大小不敏感（~30KB gzip）

---

## 架構與檔案結構

```
frontend/src/
├── i18n/
│   ├── index.ts          # i18next 初始化設定
│   ├── types.ts          # 翻譯 key 型別（確保兩份語言檔 key 一致）
│   └── locales/
│       ├── zh-TW.ts      # 繁中翻譯
│       └── en.ts         # 英文翻譯
```

### 初始化流程

1. `i18n/index.ts` 在 `main.tsx` 最頂端 import，執行一次
2. 從 `localStorage.getItem('language')` 讀取初始語言，fallback 到 `'zh-TW'`
3. 各 component 用 `const { t } = useTranslation()` 取用翻譯，無需額外 Provider

### 翻譯 key 策略

扁平化命名，以頁面為前綴，不使用 namespace 分割：

```
nav.*        — 側邊欄導覽項目
dashboard.*  — Dashboard 頁面
rules.*      — Rules 頁面
logs.*       — Logs 頁面
settings.*   — Settings 頁面
license.*    — License 相關文字
common.*     — 共用字串（loading、cancel、save…）
```

## 語言切換器 UI

新增一個 card section 在 Settings 頁「通知」區塊**上方**：

```tsx
<div className="card divide-y divide-border">
  <div className="px-5 py-4">
    <h2 className="text-sm font-semibold text-slate-900">{t('settings.language')}</h2>
  </div>
  <div className="px-5 py-4 flex items-center justify-between">
    <p className="text-sm font-medium text-slate-800">{t('settings.interfaceLanguage')}</p>
    <select
      className="input max-w-[160px]"
      value={i18n.language}
      onChange={(e) => handleLanguageChange(e.target.value)}
    >
      <option value="zh-TW">繁體中文</option>
      <option value="en">English</option>
    </select>
  </div>
</div>
```

語言切換邏輯：

```ts
function handleLanguageChange(lang: string) {
  i18n.changeLanguage(lang)
  localStorage.setItem('language', lang)
}
```

## 不在範圍內

- **日期格式**：`formatTime()` 目前 hardcode `'zh-TW'` locale，本次不改（可列為後續 nice-to-have）
- **後端字串**：`ruleName`、`errorMessage` 等來自 Go backend 的動態資料，不在前端 i18n 範圍
- **第三語言**：架構支援，但本次只實作中文和英文
